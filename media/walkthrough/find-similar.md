# Find similar code from a selection

OwlSpotlight can search using existing code as the query.

- **Selection**: highlight any code, right-click and choose
  **Find Similar Code to Selection**.
- **CodeLens**: each function definition shows a **🦉 Find similar** link. Click
  it to retrieve semantically related functions in the workspace.
- **From results**: every result row has a **🦉 Find similar** button that
  triggers a new search seeded with that function.

This is powered by the same embedding model as natural-language search.
