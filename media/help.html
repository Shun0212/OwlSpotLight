OwlSpotlight Help / ヘルプ

How to Use (English)

  • Use the Command Palette (Cmd+Shift+P or Ctrl+Shift+P) and type OwlSpotlight to access all features.
  • Start Server: OwlSpotlight: Start Server
  • Stop Server: OwlSpotlight: Stop Server
  • Clear Cache: OwlSpotlight: Clear Cache
  • Setup Python Environment: OwlSpotlight: Setup Python Environment
  • PyTorch setup: GPU auto-detect is recommended. OwlSpotlight picks CUDA 12.8 or 12.4 when an NVIDIA GPU is detected, and falls back to CPU otherwise.
  • All main actions are available from the Command Palette. You can also use the sidebar buttons for quick access. If the default port is busy, OwlSpotlight will use the next free local port automatically.

使い方 (日本語)

  • コマンドパレット（Cmd+Shift+P または Ctrl+Shift+P）で OwlSpotlight と入力すると、すべての機能が利用できます。
  • サーバー起動: OwlSpotlight: Start Server
  • サーバー停止: OwlSpotlight: Stop Server
  • キャッシュ削除: OwlSpotlight: Clear Cache
  • Python環境セットアップ: OwlSpotlight: Setup Python Environment
  • PyTorch セットアップ: GPU 自動判定がおすすめです。NVIDIA GPU が見つかれば CUDA 12.8 / 12.4 を選び、見つからなければ CPU に切り替えます。
  • 主な操作はコマンドパレットから実行できます。サイドバーのボタンもご利用ください。既定のポートが使用中なら、自動で次の空きローカルポートを使います。

Translation Settings / 翻訳設定

  • Gemini API Translation: To use Google Gemini for translation, configure in VS Code Settings:
    owlspotlight.translationSettings.geminiApiKey to your Gemini API key
    owlspotlight.translationSettings.enableJapaneseTranslation to true
  • Gemini API翻訳: Google Geminiを翻訳に使用するには、VS Codeの設定で以下を設定してください:
    owlspotlight.translationSettings.geminiApiKey に Gemini APIキー を
    owlspotlight.translationSettings.enableJapaneseTranslation を true に
  • Get Gemini API Key: Visit Google AI Studio to get your free API key.
    Gemini APIキー取得: Google AI Studioで無料のAPIキーを取得できます。

Environment Setup / 環境設定

  • Python 3.11 is required. If you use pyenv (recommended):
    pyenv install 3.11.8
    pyenv global 3.11.8
    Then restart VS Code.
    Make sure python3.11 is available in your PATH.
  • 日本語: Python 3.11が必要です。pyenvを使う場合:
    pyenv install 3.11.8
    pyenv global 3.11.8
    その後VS Codeを再起動してください。
    python3.11コマンドがPATHで使えることを確認してください。
  • On Windows, use the official Python installer and add Python to PATH.
    Windowsの場合は公式インストーラーでPythonを入れ、PATHに追加してください。
